    Local Storage Concept

    Steps to Reproduce in a Browser
    • Open the HTML file in two different tabs.
    • Both tabs will display the default text: Hello!.
    • In Tab 1, click the "Update Message" button, enter a new message (e.g., "Hello, World!"), and press OK.
    • The text in Tab 1 will update immediately because of the button click.
    • Tab 2 will automatically update the message because the storage event listener detects the change.
    • Repeat the process by making changes in Tab 2, and watch the updates reflect in Tab 1.
